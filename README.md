# my_tools_installer


## What's this script doing ?

 -  It installs brew.   
 -  It installs valgrind, htop, docker, docker-machine and minikube with brew.

## How to run ?
  1.  `./my_tools_installer.sh`
  2.  Make your confirmations.
  3.  Enjoy!
## NB:
 -  All above programs will be installed in goinfre.
